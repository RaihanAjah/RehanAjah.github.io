Certification & Training

  • Training:
  • Data Analytics at Fresh Graduate Academy (FGA) - Digitalent by Kominfo

    • The Fresh Graduate Academy (FGA) Digital Talent Scholarship 2024, organized by Indonesia's Ministry of Communication and Information Technology (Kominfo), offers a Data Analytics track via Coursera. This program aims to enhance recent graduates' competencies in data analytics, preparing them for the digital era and the global job market. The curriculum covers data analytics fundamentals, statistics, data processing with Excel and SQL, programming with Python and R, and data visualization with Tableau and Power BI. It also focuses on machine learning, predictive analytics, and ethical data practices. Participants gain significant benefits, including high-quality courses, official certification, and support from experienced mentors, preparing them to become skilled data analytics professionals.

  • Machine Learning Path Student at Bangkit Academy 2022 by Google, GoTo, and Traveloka

    • Bangkit Machine Learning Program is an Independent Study Activity for Machine Learning Developers that offers individual learning and final projects in team form. Individual learning is carried out asynchronously through learning modules on Dicoding Academy and Coursera, with the opportunity to consult with experts through discussion forums. Each participant has a mentor for consultation on non-academic difficulties. This program offers one learning path, the Machine Learning Learning Path, which covers various competencies. Participants will receive a competency certificate in each class in the learning path if they successfully pass the exam/assessment. Additionally, participants are prepared to take the global TensorFlow Developer certification exam from Google after completing this program. The material studied includes the main concepts and applications of AI to solve various ML problems with the following specializations: IT Automation with Python, Google Data Analysis, Mathematics for Machine Learning & Data Science, Machine Learning Specialization, TensorFlow Developer Professional Certificate, ML Project Setup, and TensorFlow: Data Specialization and Deployment.
